Notifications
=============

Handles in-app notifications for users such as budget alerts and goal updates.
Unlike every other feature in the application, notifications are stored in
**MongoDB Atlas**  rather than PostgreSQL, using the ``notifications`` collection.

.. note::
   Fetching notifications only returns **unread** notifications —
   ``isRead: false`` is applied as a filter in the query.

----------------

Schema
------

Notifications are stored as documents in the ``notifications`` MongoDB collection.
There is no formal schema enforced at the database level — the structure is
defined by what the service inserts.

.. code-block:: javascript

   {
     _id:       ObjectId,       // auto-generated by MongoDB
     userId:    Integer,        // references the PostgreSQL users table
     type:      String,         // e.g. "info", "warning", "alert"
     message:   String,         // notification body text
     isRead:    Boolean,        // defaults to false on creation
     createdAt: String          // ISO timestamp set on insert
   }

-------------

Routes (``routes/v1/notificationRoutes.js``)
---------------------------------------------

Four routes covering create, fetch, mark as read, and delete.
No ``authenticate`` middleware is applied.

.. code-block:: javascript

   router.post('/',                      addNotification);
   router.get('/:userId',               getNotifications);
   router.put('/:notificationId/read',  markNotificationRead);
   router.delete('/:notificationId',    removeNotification);

-------------------------

Controller (``controllers/notificationControllers.js``)
--------------------------------------------------------

Validates request fields and delegates to the service layer via
``ResponseHandler``.

``addNotification``
~~~~~~~~~~~~~~~~~~~
Extracts ``userId``, ``type``, and ``message`` from ``req.body``.
Returns ``400`` if any field is missing.

.. code-block:: javascript

   export async function addNotification(req, res) {
     try {
       const { userId, type, message } = req.body;
       if (!userId || !type || !message) {
         return ResponseHandler.badRequest(res, "userId, type, and message are required fields");
       }
       const newNotification = await createNotification(userId, type, message);
       if (newNotification) return ResponseHandler.success(res, newNotification);
       return ResponseHandler.error(res);
     } catch (error) {
       return ResponseHandler.serverError(res, error);
     }
   }

``getNotifications``
~~~~~~~~~~~~~~~~~~~~
Extracts ``userId`` from ``req.params`` and returns all unread notifications
for that user, ordered newest first.

.. code-block:: javascript

   export async function getNotifications(req, res) {
     try {
       const { userId } = req.params;
       if (!userId) return ResponseHandler.badRequest(res, "userId is required");
       const notifications = await getNotificationsByUserId(userId);
       if (notifications) return ResponseHandler.success(res, notifications);
       return res.status(400).json({ message: "Error fetching notifications" });
     } catch (error) {
       return ResponseHandler.serverError(res, error);
     }
   }

``markNotificationRead``
~~~~~~~~~~~~~~~~~~~~~~~~
Extracts ``notificationId`` from ``req.params`` and sets ``isRead: true``
on the matching document.

.. code-block:: javascript

   export async function markNotificationRead(req, res) {
     try {
       const { notificationId } = req.params;
       if (!notificationId) return ResponseHandler.badRequest(res, "notificationId is required");
       const updatedNotification = await markAsRead(notificationId);
       if (updatedNotification) return ResponseHandler.success(res, updatedNotification);
       return ResponseHandler.error(res, "Notification not found or could not be updated");
     } catch (error) {
       return ResponseHandler.serverError(res, error);
     }
   }

``removeNotification``
~~~~~~~~~~~~~~~~~~~~~~
Extracts ``notificationId`` from ``req.params`` and deletes the matching
document, returning the deleted record on success.

.. code-block:: javascript

   export async function removeNotification(req, res) {
     try {
       const { notificationId } = req.params;
       if (!notificationId) return ResponseHandler.badRequest(res, "notificationId is required");
       const deletedNotification = await deleteNotification(notificationId);
       if (deletedNotification) return ResponseHandler.success(res, deletedNotification);
       return ResponseHandler.error(res, "Notification not found or could not be deleted");
     } catch (error) {
       return ResponseHandler.serverError(res, error);
     }
   }

-----------------------

Service (``services/notificationServices.js``)
-----------------------------------------------

All database interaction using the MongoDB client retrieved via ``getNoSql()``.

``createNotification(userId, type, message)``
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
Constructs a notification document with ``isRead: false`` and an ISO
timestamp, inserts it into the collection, and returns the document with
its generated ``_id``.

.. code-block:: javascript

   export async function createNotification(userId, type, message) {
     try {
       const db = getNoSql();
       const notification = {
         userId, type, message,
         isRead: false,
         createdAt: new Date().toISOString()
       };
       const result = await db.collection(COLLECTION).insertOne(notification);
       return { _id: result.insertedId, ...notification };
     } catch (error) {
       throw error;
     }
   }

``getNotificationsByUserId(userId)``
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
Queries the collection for all documents matching ``userId`` where
``isRead`` is ``false``, sorted by ``createdAt`` descending.

.. code-block:: javascript

   export async function getNotificationsByUserId(userId) {
     try {
       const db = getNoSql();
       const notifications = await db.collection(COLLECTION)
         .find({ userId, isRead: false })
         .sort({ createdAt: -1 })
         .toArray();
       return notifications;
     } catch (error) {
       throw error;
     }
   }

``markAsRead(notificationId)``
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
Finds the document by ``ObjectId`` and sets ``isRead: true`` using
``findOneAndUpdate`` with ``returnDocument: 'after'``, returning the
updated document.

.. code-block:: javascript

   export async function markAsRead(notificationId) {
     try {
       const db = getNoSql();
       const result = await db.collection(COLLECTION).findOneAndUpdate(
         { _id: new ObjectId(notificationId) },
         { $set: { isRead: true } },
         { returnDocument: 'after' }
       );
       return result;
     } catch (error) {
       throw error;
     }
   }

``deleteNotification(notificationId)``
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
Finds and deletes the document by ``ObjectId`` in a single operation using
``findOneAndDelete``, returning the deleted document.

.. code-block:: javascript

   export async function deleteNotification(notificationId) {
     try {
       const db = getNoSql();
       const result = await db.collection(COLLECTION).findOneAndDelete({
         _id: new ObjectId(notificationId)
       });
       return result;
     } catch (error) {
       throw error;
     }
   }
