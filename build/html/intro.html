Student Finance Tracker¶

Welcome to the project documentation.

Overview¶

Student Finance Tracker (SFT) is a finance management app built specifically for students.

The app helps students take control of their money by providing tools to track spending, set savings goals, and analyse budgets using a machine learning algorithm.

To keep things engaging, SFT gamifies personal finance — earning XP points and levelling up by hitting savings goals and staying consistent.
